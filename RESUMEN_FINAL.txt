TIP 1: Cambiar colores
  → Abre src/styles/globals.css
  → Edita :root (líneas 1-20)
  → Guarda y recarga (F5)

TIP 2: Cambiar teléfono/email
  → Abre src/data/events.json
  → Busca "company"
  → Edita valores
  → Recarga página

TIP 3: Ver cambios sin código
  → Abre src/data/events.json en editor de texto
  → Busca "bautizos" → edita gallery
  → Cambia "src" para apuntar a nuevas imágenes

TIP 4: Forzar recarga del navegador
  → Windows/Linux: Ctrl+Shift+R
  → Mac: Cmd+Shift+R

TIP 5: Optimizar imágenes más
  → Usa TinyPNG (online)
  → O ImageOptim (Mac) / FileOptimizer (Windows)
